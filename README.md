# if-agents

### Requirements

Install `jericho` `dspy-ai`, `tqdm`.

### Setup

#### Acquire games
```
mkdir data
cd data
wget https://github.com/BYU-PCCL/z-machine-games/archive/master.zip
unzip master.zip
```

### Create a Config File

Create `config.json` in the base directory with

```
{"TOGETHER_API_KEY": "YOUR_API_KEY_HERE"}
```
